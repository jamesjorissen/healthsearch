Project Description

This application is designed to help people with medical needs make educated decisions about where they receive their care. By submitting the requested data in the application, users are able to find healthcare provider information in their location, anywhere in the United States. User friendliness, practicality and functionality are at the core of this application. This application pulls data from separate APIs and geolocates in real time. Enjoy!
